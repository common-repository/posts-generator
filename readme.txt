### General Feature
- Accurate or random quantities can be generated.
- It is compatible with the entire wordpress core.
- Type of deletion for data generated only by this plugin as well as all wordpress data (general reset).
- Meta data on all objects.
- Management of creation dates.

### Objects Generated
* Posts
* Users
* Comments
* Categories
* Tags
* Menus
* Meta data

___

## 📝 Posts
You can generate tens or hundreds of posts with some characteristics, such as what is the status of posts, as well as whether or not comments are allowed.
Also features of generating images inserted in the posts or featured image can also generate both.
Within the text you can generate all Inline tags and Block type tags.

## 📌 Categories
You can generate categories (hierarchy taxonomy), these terms or words are real to make more sense. You can add a description to the category. It also supports Meta data.

## 🔖 Tags
You can generate tags (non-hierarchical taxonomy), these terms or words are real to make more sense. It also supports Meta data.

## 👥 Users
You can generate more than a million combinations of names and surnames, the names are real. You can also generate types of roles by users as well as Meta data.

## 💬 Comments
It is recommended that when you generate comments you have already generated first the posts. Comment size is customizable as well as Meta data.

## 📑 Menus
You can generate menus in any position that your Theme allows, you can generate single level menus up to multilevels that is menus with submenus.

## 🗂️ Meta data
Each object that is generated also supports Meta data, these are extra data that are saved and linked to the object. By generating the Themes and Pluings use it to store more information. The available metadata are:
🔹 Text
🔹 Words
🔹 IP
🔹 Domain
🔹 HTML
🔹 Element
🔹 Long Text
🔹 Persons name
🔹 Date

## ⛓️ Integrations
Now it integrates with other plugins to generate generic content for testing. As an initial form it has been integrated with [Yuzo plugin](https://wordpress.org/plugins/yuzo-related-post/) to generate visits and generic Clicks to perform tests
